# Constitution Command

You will guide the user to create or update the `memory/constitution.md` file.

1.  **Check Existence**: Check if `memory/constitution.md` exists.
2.  **Propose Principles**: If it doesn't exist, propose a set of core principles for spec-driven development, such as:
    - Spec-First Development
    - Clear and Testable Specifications
    - Design Before Implementation
    - Task-Driven Workflow
    - Documentation and Decision Logging
3.  **Update**: If it exists, ask the user what they want to update and propose changes.
4.  **Generate File**: Create or update the `memory/constitution.md` file with the agreed-upon principles.
