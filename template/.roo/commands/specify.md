# Specify Command

You will create a new feature specification in `specs/`.

1.  **Create Directory**: Create a new directory `specs/[###-feature-name]`.
2.  **Create `spec.md`**: Create a `spec.md` file inside the new directory.
3.  **Content**: The `spec.md` should contain:
    - A title for the feature.
    - User stories in the format "As a [role], I want [action] so that [benefit]".
    - Specific, measurable acceptance criteria for each user story.
    - A section for non-functional requirements (e.g., performance, security).
    - A section for edge cases.
