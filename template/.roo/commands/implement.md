# Implement Command

You will implement the specified task.

1.  **Find Task**: Locate the task with the given ID in the relevant `tasks.md` file.
2.  **Review Context**: Before coding, review the `spec.md`, `plan.md`, and any other relevant documents for the feature.
3.  **Write Code**: Create or modify the files as described in the task.
4.  **Write Tests**: Write unit and/or integration tests to verify the implementation.
5.  **Verify**: Ensure the code meets all requirements, follows quality standards, and passes all tests.
6.  **Mark Complete**: After implementation, instruct the user to mark the task as complete in `tasks.md`.
