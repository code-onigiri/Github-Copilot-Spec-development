# Review Command

You will perform a thorough code review.

1.  **Identify Scope**: Determine the scope of the review. If a file or feature name is provided, focus on that. If not, ask the user what should be reviewed.
2.  **Activate Review Mode**: Switch to the "Code Review Mode" to use its specialized instructions and expertise.
3.  **Follow Review Process**: Systematically review the code based on the process defined in the review mode's rules:
    - Spec Compliance
    - Code Quality
    - Security
    - Performance
    - Testing
4.  **Provide Feedback**: Present the review findings in a structured report, categorizing issues by severity (Critical, High, Medium) and including positive feedback.
